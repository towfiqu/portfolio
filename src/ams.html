Project Details

A Full Stack Web Application Made with React, NodeJS & MongoDB.

Overview

Advanced Medical Subsidiary is a web app for finding nearest specialist doctors, booking appointments with doctors, booking cabins for sick patients, Creating Patients own medical profile with previous reports, detailed doctor profiles and many more.

Features

  • → Searching medical facilities
  • → doctors Search by specialty
  • → Finding facilities with google map
  • → Booking Appointment with email confirmation
  • → Booking Cabin with credit card payment
  • → Patient Profile with medical reports

Tech used for important features

Database

MongoDB is used for storing all the data. MongoDB is a very popular cross-platform NoSQL database.

Email

Emails are sent to MailTrap.io a real time development mail server for sending Email.

Email

Emails are sent to MailTrap.io a real time development mail server for sending Email.

Location

Google Maps added to show accurate locations and provide pinpoint direction to the facility.

Performance Boost

Pagination added with client side caching for faster data loading and smooth performance.

Security

Sensitive password are encrypted with bcrypt.js. A Popular NodeJS package for encrypting password. Authentication are handled with JWT to prevent any kind of route related security breach.

Payment

Payments are done through Stripe. A very Popular Payment Platform for doing any kind internet business.
