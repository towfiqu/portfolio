Project Details

A Browser Extension Made with React and third party API for Anime fans.

Overview

A browser extension that will help you keep track of animes airing on a given day and that week. Also adding a redirectable streaming link to the ones you watch.

Features

  • ⇒ Filter Animes By Day
  • ⇒ Adding Streamable Link
  • ⇒ Updating & Removing Streamable Link
  • ⇒ Redirect to your favourite Streaming Site from Extension
